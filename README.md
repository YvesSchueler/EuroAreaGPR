# Euro Area Geopolitical Risk (EA-GPR) Index

This repository provides the Euro Area Geopolitical Risk (EA-GPR) Index and related indicators developed in:

Bondarenko, Kang, Lewis, Rottner and Schüler (2026).

## Latest Data

The latest release can be downloaded here:

* **[EA_GPR_Paper.xlsx](EA_GPR_Paper.xlsx)** (latest version)

## Available Indicators

* Euro Area Geopolitical Risk (EA-GPR, monthly and daily frequency)
* Euro Area Sanctions Index
* Euro Area Shortages Index
* Country-specific indices

## Data Vintages

Historical releases are stored in the folder:

```
vintages/
```

## Citation

If you use the data, please cite:

Bondarenko, Y., Kang, N., Lewis, V., Rottner, M., and Schüler, Y. (2026): Geopolitical Risk in the Euro Area: Measurement and Transmission, Deutsche Bundesbank Discussion Paper 14/2026.
